Para poder jugar solamente debes introducir un número de tres carácteres en el recuadro, para comprobar si tu número es el correcto:
- Pulsa el botón
- Dale a la tecla enter

Si tu número es correcto te aparecerá un mensaje de que has ganado.

En cambio si tu numero es incorrecto aparecerá un mensaje de derrota.

Para volver a jugar solamente dale al botón de jugar de nuevo y listo.
